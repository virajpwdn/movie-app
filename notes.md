# For Building this app we will be using following packages

- nativewind
- tailwindcss
- react-native-reanimated
- react-native-safe-area-content

## After Installing the above pacakges, setup tailwind and nativewind

1. npx tailwind init - generates config file
2. global.css - import tailwind utilities
3. babel.config.js - a config file for babel
3. npx expo customize metro.config.js
